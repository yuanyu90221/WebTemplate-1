打開 View -> Gradle Tasks

在 bootRepackage 上按右鍵 -> Run Gradle Tasks

在 project 目錄所在地方會產生 build 資料夾, 複製 libs 底下的 jar 檔出來,

到任何地方執行

	java -jar xxx.jar 
	
即可運行

------------------------------------------------------------------------------------------

如果要用 ext {} 來決定 version, 切記 dependience 要用 " 不可以用 ' , 不然會失效
